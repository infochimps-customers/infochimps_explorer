h1. ApeyEye Explorer

An explorer for the "Infochimps.org":http://infochimps.org API.

h2. Starting this party

<pre>
    cd infochimps_explorer
    ruby ./config/bootstrap.rb
    shotgun --server=thin
</pre>

Try a describe route:
    
      http://127.0.0.1:9393/describe/engineering/chemical/msds/hazard_msds/msds_search

there's a debug-my-app route if that's helpful:
    
      http://127.0.0.1:9393/debug
